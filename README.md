# JobSeeker-App
шаблон сайта по поиску вакансий

## Установка и запуск

Клонируйте репозиторий 
```
git clone https://github.com/Gustavs3n22/JobSeeker-App.git
```

Запустите проект
```
docker compose up --build
```

Перейдите на http://localhost:8000
